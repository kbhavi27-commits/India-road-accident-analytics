## Problem Statement
Analyze India road accident data to identify high-risk zones,
peak accident times and major causes to support policy decisions.

## Dataset
- Source: Kaggle
- Records: 3,000 rows | 24 columns
- Time Period: 2018-2023

## Tools Used
Excel | MySQL Workbench | Power BI Desktop

## Key Insights
1. 50% of accidents involve alcohol
2. October is deadliest month (672 deaths)
3. State Highways more deadly than National Highways
4. Valid license drivers cause most accidents
5. Rainy weather causes highest fatalities (1,614)
